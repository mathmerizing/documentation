NumNeurNet.dataset module¶

class NumNeurNet.dataset.Dataset(name, train_size, test_size, batch_size, redownload=False)¶

Bases: object

Dataset is the class to load and interact with different Datasets. Once a dataset is loaded, it stores train_data, train_label, test_data, test_label and train_size and test_size. Formatting the data is done when loading a specific dataset and is handeled by datasets/dataset/dataset_functions.py Dataset also loads a specific print method defined in datasets/dataset/dataset_functions.py Download information is centrally stored in datasets/sources.txt and can be modified by this classes download method.
